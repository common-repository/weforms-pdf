A simple plugin to download weforms data as pdf

== Description ==

weForms-pdf will make easy to download <a href="https://wordpress.org/plugins/weforms/">weForms</a> data as pdf

You must install <a href="https://wordpress.org/plugins/weforms/">weForms</a> before using the plugin.
